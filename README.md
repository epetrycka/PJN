# Przetwarzanie języka naturalnego — projekty

Repozytorium zawiera dwa projekty z zajęć PJN:

## Projekt 1 — analiza języka portugalskiego

- Statystyczna analiza korpusu (częstości, Zipf, rdzeń języka).
- Wizualizacje i tabele w aplikacji webowej.
- Zrealizowane zadania: 1 i 3.

[Zobacz online](https://pjn-smoky.vercel.app)

## Projekt 2 — generator zdań (SVO)

- Budowanie zdań po angielsku w oparciu o Subject–Verb–Object.
- Dobór czasu, trybu, liczby oraz rodzajników.
- Zrealizowane zadanie: 2.

[Zobacz online](https://pjn-2.vercel.app)
