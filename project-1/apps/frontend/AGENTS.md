# AGENTS.md

* Będziesz tworzyć projekt o analizie statystycznej języka.
* W końcowej wersji projektu mają być dostępne poniższe funkcjonalności:
  * Tabela częstości wyrazów
  * Sprawdzenie czy występuje prawo Zipf'a
  * Rdzeń języka - graf z node'ami ustawionymi na okręgu. Rdzeniem języka będą słowa które mają najwięcej połączeń
  * Najczęściej występujące 50 rzeczowników i przetłumaczyć na polski
* Tworząc aplikację korzystaj z tailwind'a
* Tworząc komponenty używaj gotowych dostępnych w bibliotece Daisyui (Szczegóły w `./DAISYUI_AGENTS.md`)
* Nie generuj jednego wielkiego komponentu - rozdzielaj kod na mniejsze komponenty oraz wykorzystuj już stworzone wcześniej
* Oczekiwana struktura
  * screens
    * Main.tsx
    * components
      * Comp1.tsx
      * Comp2.tsx
    * ...
  * commons
    * ...
  * hooks
    * ...
  * ...
